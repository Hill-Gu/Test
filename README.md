# py-faster-rcnn
## 下载py-faster-rcnn源代码
`git clone –recursive https://github.com/rbgirshick/py-faster-rcnn.git`
## 编译caffe
### 安装依赖项
- cython
- easydict
- opencv
### 编译cython模块
```
cd py-faster-rcnn/lib
make
```
### 编译caffe和pycaffe
```
cd caffe-fast-rcnn
cp Makefile.config.example Makefile.config
gedit Makefile.config
```
